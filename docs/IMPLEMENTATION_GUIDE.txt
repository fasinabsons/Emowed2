### 3. Environment Variables

**Create `.env.local`:**
```bash

NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key

NEXT_PUBLIC_APP_URL=http://localhost:3000
```

## üêõ COMMON ISSUES & SOLUTIONS

### Issue: "Can't connect to Supabase"
```
Solution:
1. Check SUPABASE_URL is correct
2. Check SUPABASE_ANON_KEY is correct
3. Check network/firewall
4. Check Supabase project is running (not paused)
```

### Issue: "RLS policy prevents access"
```
Solution:
1. Check user is authenticated
2. Check RLS policy matches user ID
3. Use service_role key for admin operations
4. Review policies in Supabase dashboard
```

### Issue: "Slow page loads"
```
Solution:
1. Enable caching (React Query)
2. Implement pagination
3. Lazy load images
4. Use database indexes
5. Enable Supabase connection pooling
```
